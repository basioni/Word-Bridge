== Description ==

Languages Easy Strings is a plugin for WordPress. It allows to change small texts in your sites.
For example, to change the buy button title, you can change "Add to Cart" by "Buy now".

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload Easy Strings for TheCartpress to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click the new link 'discount' in the menu of TheCartPress
